# Api Load Tester Cli Tool

This is a cli tool to test load on an API.

## Usage

`lote ./path/to/json/config.json`

### Options 

- `--save file.json`    => Saves results to file
- `-q`                  => Does not pretty print, `--save` flag is required when using this
